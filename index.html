V2EX 支持的语法

✅ 支持的语法

  • 标题: # 一级标题, ## 二级标题, ### 三级标题
  • 粗体: **粗体文本** 或 __粗体文本__
  • 斜体: *斜体文本* 或 _斜体文本_
  • 删除线: ~~删除文本~~
  • 行内代码: `代码`
  • 代码块: ```语言\n代码\n```
  • 无序列表: - 或 * 或 +
  • 有序列表: 1. 2. 3.
  • 链接: [文本](URL)
  • 引用: > 引用文本
  • 分隔线: --- 或 ***

❌ 不支持的语法

  • 图片: V2EX 过滤图片标签
  • 表格: 不支持 Markdown 表格
  • 任务列表: 不支持 [ ] 和 [x]
  • 脚注: 不支持脚注功能
  • HTML 标签: 大部分 HTML 被过滤
  • 数学公式: 不支持 LaTeX 公式
